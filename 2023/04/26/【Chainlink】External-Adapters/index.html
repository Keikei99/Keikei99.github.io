什么是外部适配器

外部适配器能够实现Chainlink预言机节点本身不能实现的功能，比如对数据的加密解密等。外部适配器相当于链下API数据源到预言机节点的过渡桥梁

如何搭建外部适配器

  1. Add it to node/adapter listing service
  2. Ask a node operator to host it
  3. Run a node yourself
