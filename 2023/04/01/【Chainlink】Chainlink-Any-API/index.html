发表于2023-04-01|更新于2023-04-02

前言

在Chainlink官方文档中，Any API谈及了获取多元链下数据的方式，包括uint256，int256，bool，string，bytes32和bytes六种类型。本教程将对单元请求、多元请求、数组请求、大型请求的对应操作进行模拟。

官方文档的写法：

  • Single Word Response
  • Multi-Variable Responses
  • Array Response
  • Large Response

翻译可能不准确，不管了。

代码可以参考Make a GET Request | Chainlink Documentation

JOB的toml文件怎么写可以参考Direct Request Jobs | Chainlink Documentation
