HTML

HTML（HyperText Markup Language）是超文本标记语言，由浏览器解析生成相应的网页，是网页的骨架。

骨架

其中：

  • html是整个网页
  • head是网页的头部，用来存放给浏览器看的信息，例如CSS
  • title是网页的标题
  • body是网页的主体，用来存放给用户看的信息，例如图片、文字

标签结构

  • 标签就是在尖括号<>内放英文单词，这个英文单词称为标签名。
  • 标签分为单标签和双标签。单标签不需要考虑文本插入，比如图片img、音频audio、换行br等，故只有一个开始标签；而双标签需要考虑文本的长度，需要用开始标签和结束标签将文本括起来，比如超链接a、加粗b/strong、标题h1-h6等。
  • 两个标签之间的关系：
    • 父子（嵌套）
    • 兄弟（并列）

排版标签

标题

<h1~h6>，标题文字加粗，字号逐渐减小，且均独占一行。

段落

<p>，段落独占一行，且段落之间存在间隙。

换行和分割线

换行：br

对于行内块（inline-block）元素或者行内（inline）元素可以采用br进行换行。

分割线：hr

文本格式化

可以像word一样对文本进行样式调整。

多媒体标签

图片

图片用标签img表示，属性有以下五个：

  • src：图片url，是必须要有的属性
  • alt：替换文本，当图片加载不出来的时候要显示的文本
  • title：提示文本，鼠标悬停时会弹出提示文本
  • width：图片宽度
  • height：图片高度

注意，当只设定图片的宽度或高度时，图片会等比例进行缩放，而当宽度和高度同时设置时，可能会出现图片变形的情况。

路径

路径指的是查找文件时，从起点到终点经历的路线。路径分为绝对路径和相对路径。绝对路径一般从计算机的盘符开始，Windows的是C或D盘，MacOS和Linux一般是从根目录/开始进行寻址。相对路径是以自身文件目录作为中心进行路径寻址。

相对路径的三种情况：

  • 同一目录：./
  • 父级目录：../
  • 下级目录：./xxx

在实际的项目开发中，相对路径更加常用。

超链接

超链接用a表示，具有如下三个属性：

  • href：跳转地址，是必须的属性。开发初期如果不指定的话可以用href=“#”代替
  • target：
    • target="_blank"：在新窗口打开
    • target="_self"：覆盖本窗口（默认值）

音频

音频用audio表示，具有如下四个属性：

  • src：音频url
  • controls：音频控制面板

音频默认是没有控制面板的，需要添加controls属性才能启用

  • loop：循环播放
  • autoplay：自动播放（但是一般浏览器不允许自动播放）

视频

视频用video表示，具有如下四个属性：

  • src：视频url
  • controls：视频控制面板
  • loop：循环播放
  • autoplay：自动播放
  • muted：静音

视频的属性和音频的属性特征基本类似，但是浏览器允许视频在静音状态下自动播放。

列表

列表分为无序列表、有序列表和自定义列表。

无序列表

unorder list，用ul嵌套li。ul是无序列表，li是列表条目

  • ul标签里只能包裹li标签
  • li标签里可以包裹任何内容

有序列表

order list，用ol嵌套li。ol是有序列表，li是列表条目

  • ol标签里只能包裹li标签
  • li标签里可以包裹任何内容

自定义列表

define list，dl嵌套dt和dd，dl是定义列表，dt是定义列表的标题（table），dd是定义列表的描述/详情（description）

  • dl标签里只能包裹dt和dd标签
  • dt和dd标签里可以包裹任何内容

表格

基本使用

<table>

table嵌套tr，tr嵌套th和td。

  • table：表格
  • tr：行
  • th：表头单元格
  • td：内容单元格
  • caption标签：写在table的子级，作为表格的标题。

属性border：表格线条的宽度，目前设置为border="1"，后期样式可以用CSS进行更改

表格结构

推荐使用thead、tbody和tfoot进行标注，结构更加清晰。

  • thead：表格头部
  • tbody：表格主体
  • tfoot：表格底部

合并单元格

可以将两个相邻且内容相同的单元格合并为一个大的单元格，以合并同类信息，减少数据冗余。

步骤如下：

  1. 明确需要合并的单元格是哪两个
    • 跨列合并（水平合并）：保留最左边的单元格，添加colspan属性，属性值为合并的单元格数量
    • 跨行合并（垂直合并）：保留最上边的单元格，添加rowspan属性，属性值为合并的的单元格的数量
  3. 删除其他的单元格即可

CSS

CSS称为层叠样式表，用于对网页的装饰。

CSS的引入方式

css的引入方式分为三种：

  • 在title标签下方添加style双标签，并在style标签里面书写css代码（内部样式表，一般用于学习使用）
  • css代码写在单独的.css文件中，并在html中使用link标签进行进入（外部样式表，一般在开发项目使用）
  • 在标签中加个style属性（行内样式，配合Javascript使用）

基础选择器

标签选择器

标签选择器是使用标签命名的选择器，会选中同名标签设置相同的样式。此选择器在开发中不常用。

类选择器

类选择器选择以类定义的标签。此选择器在开发中常用，基本每个标签都需要有个class属性。

id选择器

每个标签有一个独一无二的id，id选择器将通过id选择这个单一的标签元素。id选择器一般不在css进行使用，而是在JavaScript中使用。

通配符选择器

选择所有属性。

文字控制属性

文字系列

  • 字号：font-size（数字+px）

  • 粗细：font-weight（100~900）

    • 400：正常
    • 700：加粗

  • 倾斜：font-style

    • normal：正常
    • italic：倾斜

  • 行高：line-height

    当行高属性值等于盒子高度属性值时，能够实现文字的垂直居中

  • 字体：font-family

  • font复合属性：（某个属性可以不写，但是一定要按照顺序来）

    • font-style
    • font-weight
    • size/line-height
    • font-family

水平对齐方式text-align

有三种：左（left）、中（center）、右（right）

文本缩进text-indent

一般使用em作为缩进单位（1em=1个字号大小）

文本修饰text-decooration

  • none：清除文本修饰（常用，可用于清除超链接的下划线）
  • underline：下划线
  • overline：上划线（不常用）
  • line-througn：删除线

文字颜色color

  • 半透明：rgba(r, g, b, a)
    • r：红色色值
    • g：绿色色值
    • b：蓝色色值
    • a：透明度（0~1）
  • 不透明：gba(r, g, b)
  • 默认设定颜色：red、blue、yellow

谷歌浏览器调试工具

F12，element栏可以调试html和css元素。

复合选择器

后代选择器

当需要选择父标签的某个子代标签时，可以使用后代选择器进行标签选择。

后代选择器的特点是：

  • 父代标签和子代标签之间用空格隔开
  • 将选择所有后代

子代选择器

当仅仅需要选择某个子代标签时，可以使用子代选择器进行标签选择。

子代选择器的特点是：

  • 父代标签和子代标签之间用>号隔开
  • 仅仅选择儿子标签

并集选择器

当多组标签需要设置相同的样式时，可以使用并集选择器进行标签选择。

并集选择器的特点是：

  • 每个标签用逗号隔开
  • 每个标签均具有相同的css样式

交集选择器

当需要通过标签、class、id来寻找相应html元素时，需要使用交集选择器进行选择，注意选择器之间不需要隔开。

CSS特性

继承性

继承性是指子级元素默认继承父级的文字控制属性，注意如果标签有默认文字样式则会继承失败。CSS的继承性也存在两种特殊情况：

  1. 超链接的颜色会继承失败。
  1. 标题的字体大小会继承失败

标题（比如h1-h6）的字体本身就有字体大小，如果此时修改它的字体可能会出现不可预料的后果。
